Test Instructions

Desktop Test (screen width > 1024px):

  1. Hover over "Products" menu item → Dropdown should appear
  2. Click "Products" → Should navigate to products-actuators-gearboxes.html
  3. Click dropdown links → Should navigate to respective pages
  4. Click "Home", "About", "Contact" → Should navigate immediately

Tablet/Mobile Test (screen width ≤ 1024px or use mobile menu):

  1. Click hamburger menu (☰) → Mobile menu should open
  2. Click "Products" (first time) → Dropdown should expand
  3. Click "Products" (second time) → Should navigate to products page
  4. Click dropdown links → Should navigate and close menu
  5. Click "Home", "About", "Contact" → Should navigate immediately

Back Navigation Test:

  1. Navigate to any page using the menu
  2. Click browser back button → Should return to this page
  3. Test browser forward/back buttons → Should work normally

Expected Behavior

  • ✅ Desktop: Hover shows dropdown, click navigates
  • ✅ Tablet: First click shows dropdown, second click navigates
  • ✅ Mobile menu: Hamburger opens menu, dropdowns expand/collapse correctly
  • ✅ Links without dropdowns navigate immediately on first click
  • ✅ Browser back/forward buttons work normally
  • ✅ No JavaScript console errors

Check Console for Errors

Open Developer Tools (F12) and check the Console tab. There should be no errors.

Responsive Test

Resize your browser window or use Developer Tools device emulation to test different screen sizes:

  • Desktop: > 1024px - Hover interactions
  • Tablet: 768px - 1024px - Click interactions
  • Mobile: < 768px - Hamburger menu
